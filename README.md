# 🦝 Енотов Ежедневник

Персональный планер с трекингом задач, привычек, хобби и дневником. Работает как PWA-приложение прямо в браузере телефона — установи на главный экран и пользуйся офлайн.

🔗 **Открыть приложение:** [enottail.github.io/Raccoon-s-Diary](https://enottail.github.io/Raccoon-s-Diary)

---

## ✨ Возможности

| Раздел | Описание |
|--------|----------|
| 📋 **День** | Задачи на выбранный день, прогресс-кольцо, дневник (сон, энергия, настроение, урок дня, благодарность) |
| 📅 **Неделя** | Обзор всех дней недели с прогресс-барами, быстрый переход к любому дню |
| 📊 **Аналитика** | Статистика по категориям, средний сон, архив прошлых недель |
| 💾 **Бэкап** | Экспорт и импорт данных в JSON |
| 🤖 **Telegram-бот** | Утренние и вечерние напоминания, список задач, статистика |

## 📱 Установка (PWA)

**iPhone (Safari):**
1. Открой сайт в Safari
2. Нажми **Поделиться** → **На экран "Домой"**
3. Приложение появится как иконка

**Android (Chrome):**
1. Открой сайт в Chrome
2. Нажми **⋮** → **Установить приложение**
3. Подтверди установку

## 🤖 Telegram-бот

Бот-помощник присылает уведомления и отвечает на команды.

**Команды бота:**
- `/start` — приветствие и список команд
- `/tasks` — активные задачи на сегодня
- `/tasks all` — все задачи на сегодня
- `/tasks week` — задачи на неделю
- `/week` — обзор недели с прогрессом
- `/journal` — статус дневника
- `/progress` — прогресс недели (%)
- `/stats` — полная статистика
- `/myid` — узнать свой Chat ID
- `/settings` — настройки уведомлений
- `/help` — справка

### Как привязать Telegram

1. Открой приложение → нажми 🔗 в шапке
2. Найди в Telegram бота **@Raccoons_Diary_bot**
3. Напиши ему `/myid` — он выдаст твой Chat ID
4. Вставь Chat ID в приложение → нажми **Привязать**
5. Готово! Бот будет присылать уведомления

## 🛠 Технологии

- HTML + CSS + JavaScript (ванильный, без фреймворков)
- PWA — работает офлайн, Service Worker
- Supabase — облачная синхронизация данных
- Render — хостинг Telegram-бота (Node.js + Express)
- GitHub Pages — хостинг приложения

---

## 🔄 Синхронизация

- Данные сохраняются **локально** в браузере
- При наличии интернета синхронизируются с облаком (Supabase)
- Статус синхронизации — цветная точка в шапке:
  - ⚪ Серый — офлайн
  - 🟡 Жёлтый — синхронизация
  - 🟢 Зелёный — синхронизировано
  - 🔴 Красный — ошибка
