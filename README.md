# 🦝 Енотов Ежедневник

Персональный планер с трекингом задач, привычек и дневником. PWA-приложение — работает офлайн, устанавливается на телефон.

## Возможности

- 📋 **Задачи на день** — разовые и постоянные, с чекбоксами
- 📅 **Недельный план** — обзор всей недели с прогрессом по дням
- 📊 **Аналитика** — статистика по категориям, прогресс, архив
- 📝 **Дневник дня** — сон, энергия, настроение, уроки, благодарность
- 💾 **Экспорт/Импорт** — резервное копирование в JSON
- 📱 **PWA** — устанавливается на iOS/Android, работает офлайн

## Установка на телефон

### iOS (Safari)
1. Открой сайт в Safari
2. Нажми **Поделиться** (квадрат со стрелкой)
3. Выбери **На экран «Домой»**
4. Готово! Приложение появится на главном экране

### Android (Chrome)
1. Открой сайт в Chrome
2. Нажми **⋮** → **Установить приложение**
3. Или дождись всплывающего баннера «Добавить на главный экран»

## Данные

Все данные хранятся локально в `localStorage` браузера. Для резервного копирования используй кнопку 💾 в приложении.
